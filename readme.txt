1 задание:

Реализовать загрузку данных с БД и дальнейший вывод на вью
-Создать базу данных, в ней 2 таблицы:
    1.users(айди, фио, почта, пароль)
    2.comments(айди, комментарий, дата)
Реализовать выгрузку всех данных в таблицу на вью, в таблице должны быть столбцы:
    1.айди пользователя
    2.фио
    3.комментарий
    4.дата
При выгрузке данных не использовать JOIN и выборку из двух таблиц, данные должны быть извлечены двумя отдельными запросами и связаны воедино средствами PHP
Пользователю должен соответствовать ПОСЛЕДНИЙ комментарий оставленный им.
